Compressor Plugin Demo

Compressor is a dynamic range compressor plugin that reduces the dynamic range of audio signals. It automatically reduces the volume of loud sounds or amplifies quiet sounds by narrowing the dynamic range.

This demo shows the compressor plugin hosted in a simple plugin host with level meters, LUFS monitoring, and volume control.

Parameters

  • Threshold: Level above which compression starts (dB)
  • Ratio: Amount of compression applied (e.g., 4:1)
  • Attack: How quickly compression engages (ms)
  • Release: How quickly compression disengages (ms)
  • Knee: Smoothness of the compression curve (dB)
  • Makeup Gain: Output gain compensation (dB)
