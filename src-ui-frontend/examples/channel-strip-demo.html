Channel Strip Demo

ChannelStrip is a preconfigured plugin host that contains a complete audio processing chain: EQ → Compressor → Limiter. This is a standard setup used in professional audio production for vocals, instruments, and mastering.

This demo shows how to instantiate and use the ChannelStrip component in your application. Each plugin can be selected and configured independently.

Usage

Create a ChannelStrip instance:

import { ChannelStrip } from './modules/plugins/host-channel-strip';

const container = document.getElementById('channel-strip-container');
const channelStrip = new ChannelStrip(container, {
  onPluginAdd: (plugin) => {
    console.log('Plugin added:', plugin.getName());
  },
  onPluginSelect: (plugin) => {
    console.log('Plugin selected:', plugin?.getName());
  }
});

// Access individual plugins
const eq = channelStrip.getEQ();
const compressor = channelStrip.getCompressor();
const limiter = channelStrip.getLimiter();

// Configure EQ filters
eq.addFilter({
  type: 'peak',
  frequency: 1000,
  q: 1.0,
  gain: 3.0
});

// Configure compressor
compressor.setThreshold(-20);
compressor.setRatio(4);
compressor.setAttack(10);
compressor.setRelease(100);

// Configure limiter
limiter.setThreshold(-1);
limiter.setRelease(50);

Features

  • EQ: Parametric equalizer with visual frequency response
  • Compressor: Dynamic range compression with threshold, ratio, attack, and release controls
  • Limiter: Peak limiting to prevent clipping
  • Level Meters: Input/Output monitoring with 6-channel support
  • LUFS Metering: Momentary, short-term, and integrated loudness
  • Volume Control: Master output level
  • Presets: Save and load complete channel strip configurations

Plugin Order

The plugins are arranged in the standard mastering order:

  1. EQ – Shape the frequency response before dynamics processing
  2. Compressor – Control dynamic range and add punch
  3. Limiter – Catch peaks and maximize loudness safely
