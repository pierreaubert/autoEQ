Basic EQ Demo

BasicEQ is a preconfigured plugin host that contains just an EQ plugin. It's perfect for simple equalization tasks without the complexity of a full channel strip.

This demo shows how to instantiate and use the BasicEQ component in your application.

Usage

Create a BasicEQ instance:

import { BasicEQ } from './modules/plugins/host-basic-eq';

const container = document.getElementById('basic-eq-container');
const basicEQ = new BasicEQ(container, {
  onPluginChange: (plugin) => {
    console.log('Plugin changed:', plugin.getName());
  }
});

// Access the EQ plugin
const eq = basicEQ.getEQ();

// Add a filter
eq.addFilter({
  type: 'peak',
  frequency: 1000,
  q: 1.0,
  gain: 3.0
});
